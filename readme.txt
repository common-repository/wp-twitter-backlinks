== Description ==

Plugin shows <a href="http://www.twitter.com">twitter</a> links from <a href="http://backtweets.com">backtweets.com</a> in widget area.

<a href="http://tigor.org.ua/wp-twitbacks/ ">Russian description</a>

It shows "shortened" links like bit.ly, goo.gl and other link shortering services.

You can configure your widget.

== Installation ==

1. Upload '/wp-twitbacks/' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register on <a href="http://www.backtype.com/developers">backtweets.com</a> service and get the 'API key' 
4. Configure your plugin through the 'Widgets' menu in WordPress


== Frequently Asked Questions ==

= CSS doesnot work with wp-minify activated =

Add  '/wp-content/plugins/wp-twitbacks/wp-twitbacks.css' to wp-minify exceptions.
